# 🍽️ Бот харчування для школи

Telegram-бот, через який батьки обирають варіант харчування (А або Б) на кожен день тижня.
Вибори автоматично записуються в Google Таблицю.

---

## 📋 Як виглядає для батьків

```
Батько: /start
Бот: Введіть прізвище та ім'я вашої дитини

Батько: Іваненко Петро
Бот: ✅ Реєстрація успішна! Дитина: Іваненко Петро (3-А)
     Надішліть /меню щоб обрати харчування

Батько: /меню
Бот: 📅 Понеділок, 28.04
     Оберіть варіант харчування:
     [🍱 Варіант А]  [🥗 Варіант Б]

Батько натискає: А

Бот: 📅 Вівторок, 29.04
     Оберіть варіант харчування:
     [🍱 Варіант А]  [🥗 Варіант Б]
     ... (і так для кожного дня)

Бот: 🎉 Дякуємо! Всі варіанти збережено.
```

## 🗂️ Структура Google Таблиці

### Аркуш 1: `Учні`
| A (ПІБ дитини)   | B (Клас) | C (TelegramID) |
|------------------|----------|----------------|
| Іваненко Петро   | 3-А      | _(заповнює бот)_ |
| Коваль Марія     | 3-А      |                |

> **Важливо:** Заповніть стовпці A та B заздалегідь. Стовпець C бот заповнить сам при реєстрації батьків.

### Аркуш 2: `Харчування`
| A (ПІБ)         | B (Клас) | 28.04 | 29.04 | 30.04 |
|-----------------|----------|-------|-------|-------|
| Іваненко Петро  | 3-А      | А     | Б     | А     |
| Коваль Марія    | 3-А      | Б     | А     |       |

> Дати та рядки учнів бот додає автоматично.

## 🚀 Покрокове налаштування

### Крок 1: Створити Telegram бота

1. Відкрийте Telegram, знайдіть **@BotFather**
2. Надішліть `/newbot`
3. Введіть назву: наприклад `Харчування 3-А клас`
4. Введіть username: наприклад `school3a_food_bot`
5. Скопіюйте **токен** (вигляд: `1234567890:ABCdef...`)

### Крок 2: Підключити Google Sheets

#### 2.1 Створити сервісний акаунт
1. Перейдіть на [console.cloud.google.com](https://console.cloud.google.com)
2. Створіть новий проект (або виберіть існуючий)
3. Меню → **APIs & Services** → **Enable APIs** → знайдіть `Google Sheets API` → увімкніть
4. Меню → **APIs & Services** → **Credentials**
5. **+ Create Credentials** → **Service Account**
6. Введіть будь-яке ім'я → **Create and Continue** → **Done**
7. Натисніть на створений акаунт → вкладка **Keys**
8. **Add Key** → **Create new key** → **JSON** → завантажте файл

#### 2.2 Дати доступ до таблиці
1. Відкрийте завантажений JSON-файл, знайдіть поле `client_email`
   _(виглядає як `bot-name@project.iam.gserviceaccount.com`)_
2. Відкрийте вашу Google Таблицю
3. Натисніть **Поділитися** → вставте цей email → роль **Редактор** → **Надіслати**

### Крок 3: Налаштувати аркуші таблиці
1. Перейменуйте перший аркуш на **`Учні`**
2. Рядок 1: `ПІБ дитини` | `Клас` | `TelegramID`
3. Заповніть учнів з рядка 2
4. Створіть другий аркуш з назвою **`Харчування`**
5. Рядок 1: `ПІБ дитини` | `Клас` (далі бот сам додасть дати)

### Крок 4: Розгорнути на Railway

1. Зареєструйтесь на [railway.app](https://railway.app) (через GitHub)
2. **New Project** → **Deploy from GitHub repo**
3. Підключіть цей репозиторій
4. Перейдіть у **Variables** та додайте:

| Змінна | Значення |
|--------|----------|
| `BOT_TOKEN` | Токен від BotFather |
| `SPREADSHEET_ID` | ID таблиці з URL (між `/d/` та `/edit`) |
| `GOOGLE_CREDENTIALS` | Весь вміст JSON-файлу **одним рядком** |

> 💡 Щоб перетворити JSON в один рядок: відкрийте термінал і виконайте:
> ```
> cat your-credentials.json | tr -d '\n'
> ```

5. Railway автоматично запустить бота після збереження змінних.

## ❓ Поширені питання

**Що якщо батько не знайдений у списку?**
Бот повідомить про помилку. Потрібно перевірити правопис або додати учня в аркуш `Учні`.

**Чи можна змінити вибір?**
Так — просто надішліть `/меню` знову. Нові дані перезапишуть попередні.

**Що якщо батько вибирає не в той тиждень?**
Бот завжди показує поточний тиждень (Пн–Пт). В майбутньому можна додати вибір тижня.

**Скільки коштує Railway?**
Безкоштовний план дає $5 кредитів на місяць — для невеликого бота цього достатньо.
